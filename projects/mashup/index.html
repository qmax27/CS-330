True Return Calculator

Investing in foreign markets is very complex because changing exchange rates can dramatically impact returns. This tool is designed to provide investors with an easy way to compare the true historical performance of investments made in six major stock indicies. To use the tool, simply select which country's stock index you want to invest in, choose your home currency, and specify purchase and sale dates within the last four years. This tool is uses exchange rate data from exchangeratesapi.io and stock index data from Financial Modeling Prep.
